This is my customized, feature-rich modification of the **Cosmic Bar** theme for YASB (Yet Another Status Bar). 

While keeping the original clean aesthetic as a foundation, I completely reworked the setup to add deep system monitoring, better workflow integration, and a unified UI/UX styling across all components.

## Features 

* **Komorebi Active Layout Widget:** Integrated dynamic workspace and active window layout detection.
* **GPU & System Monitor:** Replaced the stock weather widget with a custom `gpu_widget`, displaying live temperature and detailed VRAM consumption (`1.3G / 2.0G`).
* **Traffic Network Widget:** Added real-time network traffic tracking.
* **GitHub Notifications:** Integrated a persistent GitHub notification counter widget into the bar.
* **Quick Launch:** Added a dedicated quick-launch area for fast app access.
* **Unified UI/UX Enhancements:**
  * Upgraded and restyled all interface popups to match the exact theme borders and colors of the bar.
  * Custom-styled the Power-off/Session menu for a cohesive design language.

## Installation

1. Copy the configuration files into your YASB theme folder.
2. Ensure you have the required fonts/icons installed (e.g., Segoe Fluent Icons or your preferred Nerd Font).
3. Append the custom CSS classes to your local styles file.
4. Restart YASB.
